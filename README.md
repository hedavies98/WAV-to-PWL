# WAV-to-PWL
A small program to convert a WAV file to a PWL file for SPICE simulation


## Usage
Either drag the WAV file on top of the `WAV to PWL.exe` or call it from the command line with `& '.\WAV to PWL.exe' '\Path\To.WAV'`
